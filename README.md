# Plateforme-Fullstack-d-Orchestration-IA-pour-la-Classification-Zero-Shot-et-la-Synth-se-Contextuelle
Hybrid-Analyzer est une application fullstack automatisant l'analyse de veille via l'orchestration de deux IA : Hugging Face pour la classification Zero-Shot et l'API Gemini pour la génération de résumés et l'analyse de tonalité.
